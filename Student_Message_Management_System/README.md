# 学生信息管理系统 (SMMS)

## 编译和运行说明

### Windows系统（使用MinGW或Visual Studio）

#### 方法一：使用gcc命令行编译
```bash
cd coding
gcc main.c -o main.exe
main.exe
```

#### 方法二：使用Visual Studio
1. 打开Visual Studio
2. 创建新的C控制台项目
3. 将`main.c`复制到项目中
4. 按`F5`编译并运行

### Linux系统
```bash
cd coding
gcc main.c -o main -lncurses  # 需要安装ncurses库
./main
```

## 首次使用

1. 运行程序后，输入默认密码：`admin123`
2. 登录成功后，进入主菜单
3. 选择相应功能进行操作

## 功能说明

### 1. 学生成绩数据录入 (L)
- 输入学号（必须在学生信息表中存在）
- 输入课程编号、课程名称、学分
- 输入平时成绩、实验成绩（-1表示无实验）、卷面成绩
- 系统自动计算综合成绩和实得学分

### 2. 学生成绩排序 (S)
- 按综合成绩升序/降序排列
- 按实得学分升序/降序排列

### 3. 学生基本信息查询 (A)
- 按学号查询：输入学号
- 按姓名查询：输入姓名
- 按宿舍号查询：输入宿舍号（查询同一宿舍所有学生）

### 4. 学生成绩信息查询 (B)
- 按学号查询该学生的所有课程成绩
- 显示综合成绩和实得学分
- 统计总修读科目和总学分

### 5. 学生信息删除 (D)
- 输入学号删除学生
- 删除前显示学生信息确认
- 同时删除基本信息和所有成绩记录

### 6. 退出系统 (Q)
- 退出程序

## 数据文件说明

- `dataset/a.txt` - 学生基本信息（学号、姓名、性别、宿舍号、电话号码）
- `dataset/b.txt` - 学生成绩信息（学号、课程编号、课程名称、学分、各科成绩）
- `dataset/password.txt` - 登录密码文件

## 注意事项

1. 所有文本文件应使用GB2312编码以避免中文乱码
2. 删除操作不可恢复，请谨慎操作
3. 密码保存在password.txt中，可以手动修改
4. 手机号码格式验证：11位，以1开头，第二位3-9

## 常见问题

### Q: 编译时提示找不到conio.h
A: Windows系统使用MinGW或Visual Studio编译，Linux系统需要安装相应库

### Q: 中文显示乱码
A: 确保所有数据文件使用GB2312编码保存

### Q: 登录失败
A: 检查password.txt文件是否存在，默认密码为admin123
