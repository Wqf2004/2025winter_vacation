# 学生信息管理系统 - PyQt5图形化版本

## 项目简介

这是使用 PyQt5 重构的学生信息管理系统（SMMS），提供了现代化的图形用户界面。

## 功能特性

- 📝 **成绩录入**：录入学生课程成绩，自动计算综合成绩和实得学分
- 📊 **成绩排序**：支持按综合成绩和实得学分进行升序/降序排序
- 👤 **学生查询**：支持按学号、姓名、宿舍号查询学生信息
- 📈 **成绩查询**：按学号查询学生所有课程成绩
- 🗑️ **信息删除**：删除学生及其所有成绩记录
- 🔐 **登录验证**：系统登录验证，保护数据安全

## 安装说明

### 前置要求

- Python 3.7 或更高版本


## 数据文件

程序使用以下数据文件（位于 `../dataset/` 目录）：

- `a.txt` - 学生基本信息（学号、姓名、性别、宿舍号、电话号码）
- `b.txt` - 学生成绩信息（学号、课程编号、课程名称、学分、各科成绩）
- `password.txt` - 登录密码文件

## 使用说明

### 1. 登录系统

启动程序后，输入密码登录系统。默认密码为 `admin123`。

### 2. 成绩录入

1. 从学生列表中点击选择学生，或查看表格后手动输入学号
2. 填写课程编号、课程名称、学分
3. 填写平时成绩、实验成绩（无实验输入-1）、卷面成绩
4. 点击"提交"按钮保存成绩

### 3. 成绩排序

1. 选择排序方式（综合成绩升序/降序、实得学分升序/降序）
2. 点击"开始排序"按钮查看结果

### 4. 学生查询

1. 选择查询方式（按学号、姓名、宿舍号）
2. 输入查询内容
3. 点击"查询"按钮或按回车键查看结果

### 5. 成绩查询

1. 输入学号
2. 点击"查询"按钮或按回车键查看该学生所有课程成绩
3. 系统自动统计总修读科目和实得总学分

### 6. 信息删除

1. 输入要删除的学生学号
2. 点击"查询"按钮查看学生信息
3. 点击"删除该学生及其所有成绩"按钮确认删除

## 注意事项

1. 删除操作不可恢复，请谨慎操作
2. 密码保存在 `dataset/password.txt` 中，可手动修改
3. 数据文件使用 GBK 编码，请勿使用其他编码编辑
4. 手机号码格式验证：11位，以1开头，第二位为3-9

## 综合成绩计算规则

```
有实验课程：
综合成绩 = 平时成绩 × 0.15 + 实验成绩 × 0.15 + 卷面成绩 × 0.70

无实验课程：
综合成绩 = 平时成绩 × 0.30 + 卷面成绩 × 0.70
```

## 实得学分计算规则（等级学分制）

```
综合成绩 ≥ 90：实得学分 = 学分 × 1.0
综合成绩 ≥ 80：实得学分 = 学分 × 0.8
综合成绩 ≥ 70：实得学分 = 学分 × 0.75
综合成绩 ≥ 60：实得学分 = 学分 × 0.6
综合成绩 <  60：实得学分 = 0
```
