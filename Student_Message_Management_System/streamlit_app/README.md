# 学生信息管理系统 - Streamlit网页应用

## 项目简介

这是使用 Streamlit 构建的学生信息管理系统（SMMS）网页应用版本，无需安装任何软件即可在浏览器中使用。

## 功能特性

- 📝 **成绩录入**：录入学生课程成绩，自动计算综合成绩和实得学分
- 📊 **成绩排序**：支持按综合成绩和实得学分进行升序/降序排序
- 👤 **学生查询**：支持按学号、姓名、宿舍号查询学生信息
- 📈 **成绩查询**：按学号查询学生所有课程成绩
- 🗑️ **信息删除**：删除学生及其所有成绩记录
- 🔐 **登录验证**：系统登录验证，保护数据安全
- 📊 **数据统计**：实时显示学生总数、成绩记录数、平均成绩

## 安装说明

### 前置要求

- Python 3.7 或更高版本


### 启动应用

```bash
cd streamlit_app

```

启动后，浏览器会自动打开 `http://localhost:8501`

### 默认密码

## 数据文件

程序使用以下数据文件（位于 `../dataset/` 目录）：

- `a.txt` - 学生基本信息（学号、姓名、性别、宿舍号、电话号码）
- `b.txt` - 学生成绩信息（学号、课程编号、课程名称、学分、各科成绩）
- `password.txt` - 登录密码文件

## 使用说明

### 1. 登录系统

启动应用后，输入密码登录系统。默认密码为 `admin123`。

### 2. 成绩录入

1. 从下拉框选择学生学号
2. 填写课程编号、课程名称、学分
3. 填写平时成绩、实验成绩（无实验输入-1）、卷面成绩
4. 点击"提交成绩"按钮保存成绩

### 3. 成绩排序

1. 选择排序方式（综合成绩升序/降序、实得学分升序/降序）
2. 点击"开始排序"按钮查看结果
3. 结果以表格形式展示，包含学生姓名

### 4. 学生查询

1. 选择查询方式（按学号、姓名、宿舍号）
2. 输入查询内容
3. 点击"查询"按钮查看结果
4. 系统自动验证手机号码格式

### 5. 成绩查询

1. 输入学号
2. 点击"查询"按钮查看该学生所有课程成绩
3. 系统自动统计总修读科目和实得总学分

### 6. 信息删除

1. 输入要删除的学生学号
2. 点击"查询学生信息"按钮查看详细信息
3. 确认无误后点击"删除该学生及其所有成绩"按钮

## 注意事项

1. 删除操作不可恢复，请谨慎操作
2. 密码保存在 `dataset/password.txt` 中，可手动修改
3. 数据文件使用 GBK 编码，请勿使用其他编码编辑
4. 手机号码格式验证：11位，以1开头，第二位为3-9
5. 多人同时使用时，请确保数据文件不被锁定

## 综合成绩计算规则

```
有实验课程：
综合成绩 = 平时成绩 × 0.15 + 实验成绩 × 0.15 + 卷面成绩 × 0.70

无实验课程：
综合成绩 = 平时成绩 × 0.30 + 卷面成绩 × 0.70
```

## 实得学分计算规则（等级学分制）

```
综合成绩 ≥ 90：实得学分 = 学分 × 1.0
综合成绩 ≥ 80：实得学分 = 学分 × 0.8
综合成绩 ≥ 70：实得学分 = 学分 × 0.75
综合成绩 ≥ 60：实得学分 = 学分 × 0.6
综合成绩 <  60：实得学分 = 0
```

## 常见问题

**Q: 如何修改端口？**
```bash
streamlit run app.py --server.port 8080
```

**Q: 如何禁用缓存？**
在 `app.py` 中移除 `@st.cache_resource` 装饰器

**Q: 如何自定义主题？**
创建 `.streamlit/config.toml` 文件配置主题

**Q: 中文显示乱码怎么办？**
确保数据文件使用 GBK 编码保存
