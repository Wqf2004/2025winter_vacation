## 数据验证功能

### 手机号码验证

有效手机号码格式：
- ✅ 11位数字
- ✅ 以1开头
- ✅ 第二位为3-9

示例：
- 有效：18376304063 ✓
- 无效：12345678901 ✗（第二位不是3-9）
- 无效：1837630406 ✗（不是11位）

### 学号验证

- 验证学号是否存在于`dataset/a.txt`中
- 录入成绩时自动验证
- 查询时自动验证

## 综合成绩计算规则

```
有实验课程：
综合成绩 = 平时成绩 × 0.15 + 实验成绩 × 0.15 + 卷面成绩 × 0.70

无实验课程：
综合成绩 = 平时成绩 × 0.30 + 卷面成绩 × 0.70
```

## 实得学分计算规则（等级学分制）

```
综合成绩 ≥ 90：实得学分 = 学分 × 1.0
综合成绩 ≥ 80：实得学分 = 学分 × 0.8
综合成绩 ≥ 70：实得学分 = 学分 × 0.75
综合成绩 ≥ 60：实得学分 = 学分 × 0.6
综合成绩 <  60：实得学分 = 0
```

## 常见问题

**Q: 如何修改密码？**
A: 直接编辑`dataset/password.txt`文件，修改为新的密码即可。

**Q: 删除的数据可以恢复吗？**
A: 不可以，删除操作不可恢复，请谨慎操作。

**Q: 支持批量录入吗？**
A: 当前版本不支持批量录入，需要逐条录入。

**Q: 可以修改已有的学生信息吗？**
A: 当前版本不支持修改功能，如需修改需要删除后重新录入。

**Q: 为什么中文显示乱码？**
A: 确保所有数据文件使用GB2312编码保存。
