# 矩阵运算系统 - PyQt5桌面版

## 功能特性

✅ **矩阵加法** - 两个相同维度的矩阵相加
✅ **矩阵减法** - 两个相同维度的矩阵相减
✅ **矩阵乘法** - 矩阵乘法（满足行×列规则）
✅ **矩阵转置** - 将矩阵行列互换
✅ **矩阵求逆** - 使用NumPy计算逆矩阵
✅ **矩阵三角化** - 使用高斯消元法转换为上三角形式

## 界面特点

- 📊 直观的表格输入界面
- 🔢 可自定义矩阵尺寸（最大20×20）
- 💡 实时运算提示和错误检查
- 🎨 现代化的用户界面设计
- ⚡ 快速计算，实时显示结果

## 安装和运行

### Windows系统

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
run.bat
```

或直接运行：
```bash
python main.py
```

### Linux/Mac系统

1. 添加执行权限：
```bash
chmod +x run.sh
```

2. 运行程序：
```bash
./run.sh
```

或直接运行：
```bash
python3 main.py
```

## 使用方法

1. **选择运算类型**：从下拉菜单选择要执行的运算
2. **输入矩阵**：在表格中直接输入数字，或修改行数和列数
3. **点击计算**：点击"计算"按钮执行运算
4. **查看结果**：结果将显示在下方的结果表格中
5. **清空**：点击"清空"按钮重新开始

## 运算规则说明

### 矩阵加法/减法
- 两个矩阵必须具有相同的行数和列数
- 结果矩阵的维度与输入矩阵相同

### 矩阵乘法
- 第一个矩阵的列数必须等于第二个矩阵的行数
- 结果矩阵的行数等于第一个矩阵的行数
- 结果矩阵的列数等于第二个矩阵的列数

### 矩阵转置
- 任何矩阵都可以转置
- 结果矩阵的行数=原矩阵的列数
- 结果矩阵的列数=原矩阵的行数

### 矩阵求逆
- 仅适用于方阵（行数=列数）
- 矩阵的行列式不能为零（可逆）
- 逆矩阵与原矩阵相乘等于单位矩阵

### 矩阵三角化
- 将矩阵转换为上三角形式
- 使用高斯消元法实现
- 对角线以下元素为零

## 注意事项

1. 矩阵尺寸限制：最大支持20×20
2. 数值精度：结果显示保留6位有效数字
3. 输入格式：支持整数和小数
4. 求逆运算：仅对方阵有效，且需行列式非零
5. 不可逆矩阵：系统会提示错误信息

## 故障排除

### 程序无法启动

1. 检查Python版本（需3.7+）
2. 确认已安装依赖：`pip list`
3. 重新安装依赖：`pip install -r requirements.txt`

### 矩阵运算错误

1. 检查矩阵维度是否符合运算规则
2. 确认输入的数值格式正确
3. 查看系统提示的错误信息
