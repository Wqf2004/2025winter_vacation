# 矩阵运算系统 - Streamlit网页版

## 功能特性

✅ **矩阵加法** - 两个相同维度的矩阵相加
✅ **矩阵减法** - 两个相同维度的矩阵相减
✅ **矩阵乘法** - 矩阵乘法（满足行×列规则）
✅ **矩阵转置** - 将矩阵行列互换
✅ **矩阵求逆** - 使用NumPy计算逆矩阵
✅ **矩阵三角化** - 使用高斯消元法转换为上三角形式

## 界面特点

- 🌐 基于Web的交互界面
- 📊 响应式布局，支持移动端
- 💡 实时运算提示和错误检查
- 🎨 现代化的UI设计
- 🔄 自动联调矩阵尺寸

## 安装和运行

### Windows系统

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
run.bat
```

或直接运行：
```bash
streamlit run app.py
```

### Linux/Mac系统

1. 添加执行权限：
```bash
chmod +x run.sh
```

2. 运行程序：
```bash
./run.sh
```

或直接运行：
```bash
streamlit run app.py
```

## 使用方法

### 1. 选择运算类型
在左侧边栏选择要执行的运算类型：
- 矩阵加法
- 矩阵减法
- 矩阵乘法
- 矩阵转置
- 矩阵求逆
- 矩阵三角化

### 2. 设置矩阵尺寸
- 输入行数和列数（最大20×20）
- 系统会自动联调第二个矩阵的尺寸以符合运算规则

### 3. 输入矩阵元素
在文本框中输入矩阵元素：
- 每行输入一行数值
- 用空格或逗号分隔
- 例如：
  ```
  1 2 3
  4 5 6
  7 8 9
  ```

### 4. 点击计算
点击"🧮 计算"按钮执行运算，结果会以表格形式显示

## 运算规则说明

### 矩阵加法/减法
- 两个矩阵必须具有相同的行数和列数
- 结果矩阵的维度与输入矩阵相同
- 系统自动设置矩阵2的尺寸与矩阵1相同

### 矩阵乘法
- 第一个矩阵的列数必须等于第二个矩阵的行数
- 结果矩阵的行数等于第一个矩阵的行数
- 结果矩阵的列数等于第二个矩阵的列数
- 系统自动设置矩阵2的行数等于矩阵1的列数

### 矩阵转置
- 任何矩阵都可以转置
- 结果矩阵的行数=原矩阵的列数
- 结果矩阵的列数=原矩阵的行数

### 矩阵求逆
- 仅适用于方阵（行数=列数）
- 矩阵的行列式不能为零（可逆）
- 逆矩阵与原矩阵相乘等于单位矩阵

### 矩阵三角化
- 将矩阵转换为上三角形式
- 使用高斯消元法实现
- 对角线以下元素为零

## 注意事项

1. 矩阵尺寸限制：最大支持20×20
2. 数值精度：结果显示保留6位有效数字
3. 输入格式：支持空格、逗号、制表符分隔
4. 求逆运算：仅对方阵有效，且需行列式非零
5. 不可逆矩阵：系统会提示错误信息
6. 自动纠错：输入格式错误时会显示详细错误信息

## 输入示例

### 示例1：矩阵加法
```
矩阵1 (2×3):
1 2 3
4 5 6

矩阵2 (2×3):
7 8 9
10 11 12

结果:
8 10 12
14 16 18
```

### 示例2：矩阵乘法
```
矩阵1 (2×3):
1 2 3
4 5 6

矩阵2 (3×2):
7 8
9 10
11 12

结果 (2×2):
58 64
139 154
```

### 示例3：矩阵求逆
```
输入矩阵 (2×2):
1 2
3 4

逆矩阵:
-2 1
1.5 -0.5
```

## 故障排除

### 程序无法启动

1. 检查Python版本（需3.7+）
2. 确认已安装依赖：`pip list`
3. 重新安装依赖：`pip install -r requirements.txt`

### 浏览器无法打开

1. 检查是否被防火墙阻止
2. 手动访问：http://localhost:8501
3. 使用不同端口：`streamlit run app.py --server.port 8502`

### 矩阵运算错误

1. 检查矩阵维度是否符合运算规则
2. 确认输入的数值格式正确
3. 查看系统提示的错误信息
