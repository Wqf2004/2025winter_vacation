# 矩阵运算系统 - C语言控制台版

## 功能说明

### 实现方式1：参数为二维数组名、行数、列数
- 矩阵加法
- 矩阵减法
- 矩阵乘法
- 矩阵转置
- 矩阵打印

### 实现方式2：参数为指针、行数、列数
- 矩阵加法
- 矩阵减法
- 矩阵乘法
- 矩阵转置
- 矩阵求逆（高斯消元法）
- 矩阵三角化（高斯消元法）

## 编译运行

### Windows
```bash
compile.bat
```

### Linux/Mac
```bash
chmod +x compile.sh
./compile.sh
```

## 技术特点

1. **两种参数传递方式**
   - 方式1：直接传递二维数组 `matrix[MAX_SIZE][MAX_SIZE]`
   - 方式2：传递指针 `double *matrix`

2. **矩阵运算功能**
   - 矩阵加、减、乘
   - 矩阵转置
   - 矩阵求逆（使用高斯消元法）
   - 矩阵三角化（使用高斯消元法）

3. **矩阵限制**
   - 最大尺寸：20×20
   - 数据类型：double（双精度浮点数）

4. **容错处理**
   - 维度匹配检查
   - 不可逆矩阵检测
   - 输入验证
