# Josephus环问题 - C语言控制台版

## 问题描述

包含四种Josephus环问题的变体实现。

## 编译和运行

### Windows
```bash
compile.bat
```

### Linux/Mac
```bash
bash compile.sh
```

## 功能

1. **带人名的Josephus环问题** - 输入n个人名，从第k个人开始，报数m，输出出列顺序
2. **反Josephus环问题** - 给定n, m, x，求使第x个人最后出列的起始位置k
3. **Josephus环问题的变形** - 2k人（前k好人后k坏人），求使最后剩下k个好人的最小m
4. **直线型报数问题** - 使用卡片数值进行报数淘汰
