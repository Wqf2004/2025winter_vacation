# Josephus环问题 - Streamlit网页版

## 问题描述

包含四种Josephus环问题的变体可视化实现。

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行

### Windows
```bash
run.bat
```

### Linux/Mac
```bash
bash run.sh
```

## 功能

1. **带人名的Josephus环问题** - 可视化模拟出列过程
2. **反Josephus环问题** - 搜索使指定人员最后出列的起始位置
3. **Josephus环问题的变形** - 寻找使好人生存的最小报数m
4. **直线型报数问题** - 可视化卡片报数淘汰过程
