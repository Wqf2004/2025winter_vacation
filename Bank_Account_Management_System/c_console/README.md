# C 控制台版 - 银行账目管理系统

## 编译运行

### Windows
```bash
# 编译
compile.bat

# 运行
main.exe
```

### Linux/Mac
```bash
# 编译
bash compile.sh

# 运行
./main
```

## 功能说明

### 登录验证
- 默认密码：admin123
- 允许三次输入机会

### 主菜单
1. 开户 - 创建新账户
2. 借款 - 借入资金（余额变负）
3. 还款 - 偿还借款
4. 存款 - 存入资金（余额变正）
5. 查询账户 - 按账号查询
6. 查询最大借款账户 - 显示借款最多的账户
7. 查询最大存款账户 - 显示存款最多的账户
8. 按借款余额排序 - 借款户按余额从大到小排序
9. 按存款余额排序 - 存款户按余额从大到小排序
10. 按开户日期排序 - 按日期升序排列
11. 清户 - 删除账户（逻辑删除）
12. 文件紧缩 - 清除已删除账户
13. 统计信息 - 显示各类统计数据

### 预警功能
- 程序启动时自动检查借款额超过5万元的账户

## 数据文件
- `../dataset/password.txt` - 密码文件
- `../dataset/AccountBasic.txt` - 账户基本信息
- `../dataset/AccountBalance.txt` - 账户余额
