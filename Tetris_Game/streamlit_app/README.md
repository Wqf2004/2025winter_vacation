# 俄罗斯方块游戏 - Streamlit网页版

## 游戏功能

- 7种经典方块形状
- Web浏览器界面
- 方块旋转、移动、加速
- 消行计分系统
- 等级系统
- 下一个方块预览
- 暂停/继续
- 重新开始

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行方式

### Windows
```bash
run.bat
```

### Linux/Mac
```bash
bash run.sh
```

## 操作说明

点击界面按钮控制游戏：
- **左移** - 方块向左移动
- **加速** - 方块加速下落
- **旋转** - 旋转方块
- **暂停/继续** - 暂停或继续游戏
- **重新开始** - 重置游戏

## 注意事项

由于Streamlit的交互限制，此版本通过点击按钮控制，而不是键盘控制。
