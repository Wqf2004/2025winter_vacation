# 俄罗斯方块游戏 - PyQt5桌面版

## 游戏功能

- 7种经典方块形状
- 流畅的图形界面
- 方块旋转、移动、加速
- 消行计分系统
- 等级系统
- 下一个方块预览
- 暂停/继续
- 重新开始

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行方式

### Windows
```bash
run.bat
```

### Linux/Mac
```bash
bash run.sh
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ | 旋转方块 |
| ← | 左移方块 |
| → | 右移方块 |
| ↓ | 加速下落 |
| 空格 | 暂停/继续 |
