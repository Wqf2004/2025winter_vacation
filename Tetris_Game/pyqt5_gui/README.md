# 俄罗斯方块游戏 - PyQt5桌面版（集成AI引擎）

## 游戏功能

- 7种经典方块形状
- 流畅的图形界面
- 方块旋转、移动、加速
- 消行计分系统
- 等级系统
- 下一个方块预览
- 暂停/继续
- 重新开始
- **AI智能辅助和对比系统**

## AI系统核心思想

### 问题建模

将每个游戏局面、当前方块、下一个方块视为一个**决策问题**：
- **输入**：当前棋盘状态、当前方块类型、下一个方块类型
- **输出**：最优的放置位置（旋转角度+横向位置）
- **评估**：通过评估函数为每个可行解打分

### 可行解空间

每个决策问题有多个可行解：
- **解的定义**：(旋转次数, 位置) 的组合
- **可行解数量**：根据方块形状去重后，通常有9-32个可行解
  - I形：17个解（2种旋转状态）
  - O形：9个解（1种旋转状态）
  - S/Z形：16个解（2种旋转状态）
  - J/L形：32个解（4种旋转状态）
  - T形：34个解（4种旋转状态，交替8/9个位置）
- **AI任务**：在所有可行解中找到得分最高的最优解

### 人类学习机制

人类玩家的选择作为**较优解的参考**：

1. **人类选择 ≠ 最优解**
   - 人类的选择通常是经验驱动的较优解
   - 未必是全局最优，但通常合理

2. **AI对比分析**
   - AI评估所有可行解并排序
   - 将人类解与AI最优解对比
   - 当AI找到更高得分的解时，视为**超人类解**

3. **持续改进**
   - 记录每次人类选择和AI建议
   - 统计AI超人类的次数和比例
   - 通过超人类解的特征分析改进评估函数

### 启发式评估体系

AI使用多维度特征评估放置解的质量：

| 特征 | 权重 | 说明 |
|------|--------|------|
| 消除行数 | +300 | 消除的行数（最重要） |
| 落点高度 | +15/行 | 方块落点高度（正向奖励，落点越低越好） |
| 空洞数量 | -20/个 | 产生的空洞数（强惩罚） |
| 最高点 | -3/行 | 棋盘最高点（越低越好） |
| 不平整度 | -1/差 | 列间高度差异 |
| 填补缺口 | +50/加权行 | 填补接近完成的行（底部权重高） |
| 方块接触评分 | +1/分 | 侧边界1分，底部3分，中间2分 |
| 下块友好度 | +10/分 | 下个方块的最佳得分潜力 |

### AI模式

#### 1. 游戏辅助模式

- AI实时分析当前局面
- 按A键显示AI建议
- 游戏界面用绿色虚线框显示建议位置
- 只显示人类可到达的位置（通过BFS路径搜索验证可达性）
- 过滤掉被方块遮挡无法进入的空缺（如拐角、狭窄通道）
- 适合普通玩家，提供实用建议

#### 2. 辅助学习模式

- AI实时分析当前局面
- 按A键显示AI建议
- 显示所有可能的放置位置（包括被遮挡的位置）
- 用于收集训练数据，帮助AI学习更优策略
- 适合高级玩家和AI训练

#### 3. AI自动游戏

- AI完全控制游戏决策
- 实时选择最优放置位置
- 支持机器学习优化

### 机器学习动态评分

#### 核心特性

- **动态权重预测**：使用神经网络根据游戏状态预测最优评分权重
- **渐进式优化**：从静态权重逐步过渡到完全学习权重
- **人类引导学习**：从人类拒绝AI建议的决策中学习
- **经验回放**：存储并重用历史数据提升训练效果

#### 训练流程

```
人类初始决策
    ↓
AI评估并生成辅助决策
    ↓
人类坚持初始决策（拒绝AI建议）
    ↓
记录差异样本到经验回放缓冲区
    ↓
训练神经网络调整权重
    ↓
逐步提升AI辅助能力
```

#### 网络架构

- **输入层（212维）**：
  - 棋盘状态：200维（20×10）
  - 当前方块：7维（one-hot编码）
  - 下一个方块：7维（one-hot编码）
  - 游戏上下文：3维（消除行数、最高点、轮次）

- **隐藏层**：
  - Dense(256) → ReLU
  - Dense(128) → ReLU
  - Dense(64) → ReLU

- **输出层（9维）**：动态预测的评分权重

#### 训练阶段

1. **静态阶段**：完全使用静态评分规则
2. **混合阶段**：静态权重与学习权重按比例融合
3. **学习阶段**：完全使用神经网络预测的权重

## 使用AI功能

### 步骤1：选择AI模式

在游戏界面右侧选择AI模式：
- **关闭**：纯手动游戏
- **AI辅助**：按A键获取AI建议，绿色虚线框显示最优位置
- **AI对比**：AI自动记录你的选择并对比分析

### 步骤2：AI辅助模式

1. 方块出现后，按A键
2. 界面显示AI建议的旋转和位置
3. 绿色虚线框显示建议的落点
4. 你可以选择跟随AI建议或按自己的想法玩

### 步骤3：AI对比模式

1. 正常游玩，AI在后台记录你的每次选择
2. 每次方块锁定时，自动弹出对比窗口
3. 查看你的选择vs AI最优解的得分差异
4. 界面右侧显示AI超人类统计

### 步骤4：分析结果

对比窗口显示：
- 你的选择：旋转、位置、评分
- AI最优解：旋转、位置、评分
- 评价：AI是否找到了更优解，提升了多少分
- 统计：AI超人类的总次数和比例

## 技术细节

### 评估函数计算

```
Score = Σ (特征值 × 权重)

例如：
Score = lines_cleared × 300
       + drop_height × (-15)
       + holes × (-8)
       + highest_point × (-6)
       + ...其他特征
```

### 可行解枚举

对于每个(旋转, 位置)组合：
1. 检查放置是否碰撞
2. 模拟放置到棋盘
3. 消除完整行
4. 提取放置后的特征
5. 计算评分
6. 按评分排序

### 超人类解判定

AI解 > 人类解 + 阈值（如10分）→ 超人类解
