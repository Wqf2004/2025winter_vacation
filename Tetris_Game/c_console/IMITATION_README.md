# 模仿学习功能使用指南

## 功能说明

模仿学习（Imitation Learning）允许AI从人类玩家的游戏记录中学习策略。

## 使用流程

### 1. 手动玩游戏并记录操作

1. 选择主菜单选项 1 - 开始游戏
2. 正常玩游戏
3. 每次放置方块时，系统会自动记录：
   - 当前方块和下一个方块类型
   - 玩家选择的旋转和位置
   - 当前的棋盘特征（高度、空洞、不平整度）
   - 消除的行数

4. 游戏结束后，记录会自动保存到 `tetris_player_records.txt`

### 2. 从玩家记录初始化AI

1. 选择主菜单选项 5 - 从玩家记录初始化AI
2. 系统会加载 `tetris_player_records.txt`
3. 使用记录初始化Q表
4. AI会学习人类玩家的策略倾向

### 3. 继续训练优化

1. 选择主菜单选项 4 - AI训练模式
2. AI会基于人类玩家的策略继续优化
3. 通过强化学习进一步改进策略

## 记录文件格式

`tetris_player_records.txt` 文件格式：

```
# 俄罗斯方块游戏记录
# 格式: current_shape next_shape rotation position height_level holes_level bumpiness_level lines_cleared
# 记录数: X

0 1 0 5 1 0 2 0
1 2 1 3 1 0 1 1
...
```

- `current_shape`: 当前方块类型 (0-6)
- `next_shape`: 下一个方块类型 (0-6)
- `rotation`: 玩家选择的旋转 (0-3)
- `position`: 玩家选择的位置 (0-9)
- `height_level`: 棋盘高度等级 (0-4)
- `holes_level`: 棋盘空洞等级 (0-3)
- `bumpiness_level`: 棋盘不平整度等级 (0-4)
- `lines_cleared`: 消除的行数

## Q值初始化逻辑（改进版）

从人类玩家记录初始化Q表时：

### 两遍扫描算法

**第一遍：统计玩家选择**
- 统计每个状态-动作对被选择的次数
- 计算每个动作的平均得分
- 分析玩家的行为模式

**第二遍：学习策略偏好**
- `Q值 = 平均奖励 + 频率奖励`
- 频率奖励：经常被选择的行为给予额外奖励
- 学习玩家的策略倾向和直觉

### 得分评估系统

评估玩家每个选择的综合得分：

1. **消除行奖励**（最重要）：
   - 消除1行：+200分
   - 消除4行（Tetris）：+300分（额外奖励）

2. **高度控制**：
   - 等级0（0-3）：+30分（优秀）
   - 等级1（4-7）：+15分（好）
   - 等级2（8-11）：+5分（一般）
   - 等级3（12-15）：-10分（差）
   - 等级4（16+）：-30分（很差）

3. **空洞避免**：
   - 0个空洞：+25分（完美）
   - 1-3个空洞：+10分（好）
   - 4-6个空洞：-10分（差）
   - 7+个空洞：-30分（很差）

4. **平整度控制**：
   - 等级0（0-3）：+20分（完美）
   - 等级1（4-7）：+10分（好）
   - 等级2（8-11）：+5分（一般）
   - 等级3（12-15）：-10分（差）
   - 等级4（16+）：-20分（很差）

5. **组合奖励**：
   - 消除行+低高度：+50分（大奖励）
   - Tetris（4行）：+100分（超大奖励）

### 学习效果

AI会学到：
- **策略偏好**：玩家在不同局面下倾向于做什么
- **得失判断**：哪些选择是好是坏
- **游戏风格**：激进/保守/冒险/稳健
- **优化方向**：玩家擅长的策略类型

## 优势

1. **快速启动**：AI无需从零开始训练
2. **人类启发**：学习人类玩家的策略直觉
3. **持续优化**：通过强化学习继续改进
4. **减少训练时间**：相比纯强化学习，收敛更快

## 编译

```batch
compile_with_ai.bat
```

这会编译以下文件：
- `main.c` - 主程序
- `tetris_ai.c` - AI核心逻辑
- `imitation_learning.c` - 模仿学习模块
