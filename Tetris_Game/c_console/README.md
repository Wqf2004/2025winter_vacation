# 俄罗斯方块游戏 - C语言控制台版

## 游戏功能

- 7种经典方块形状（I、J、L、O、S、T、Z）
- 方块旋转、左右移动、加速下落
- 消行计分系统
- 等级系统（每消除10行升一级）
- 暂停/继续功能
- 下一个方块预览

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ | 旋转方块 |
| ← | 左移方块 |
| → | 右移方块 |
| ↓ | 加速下落 |
| P | 暂停/继续 |
| Q | 退出游戏 |

## 编译和运行

### Windows
```bash
compile.bat
```

### Linux/Mac
```bash
bash compile.sh
```

## 计分规则

- 消除1行: 100 × 等级
- 消除2行: 400 × 等级
- 消除3行: 900 × 等级
- 消除4行: 1600 × 等级
