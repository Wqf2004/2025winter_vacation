# 电话簿管理系统 - C语言控制台版

## 功能说明

1. **添加联系人** - 输入姓名、工作单位、电话号码和E-mail地址
2. **删除联系人** - 按姓名查找并删除联系人
3. **显示所有联系人** - 列表形式显示所有联系人信息
4. **修改联系人** - 修改联系人的各项信息
5. **排序功能** - 按姓名字母序或按电话号码排序
6. **查询功能** - 按姓名或电话号码查询联系人
7. **保存到文件** - 将联系人信息保存到文件
8. **从文件读取** - 从文件加载联系人信息
9. **修改密码** - 修改系统登录密码
10. **数据验证** - 自动验证电话号码和E-mail格式

## 数据验证规则

### 电话号码验证
- 长度：7-15位
- 允许字符：数字、连字符(-)、空格
- 自动检查重复号码

### E-mail验证
- 长度：5-99位
- 必须包含且仅包含一个 @ 符号
- @ 符号后必须包含点号
- 格式：xxx@xxx.xxx

## 编译运行

### Windows
```bash
compile.bat
```

### Linux/Mac
```bash
chmod +x compile.sh
./compile.sh
```

## 默认密码

- 默认密码：`admin123`
- 首次使用可通过菜单选项9修改密码

## 数据文件

- `dataset/contacts.txt` - 联系人数据文件
- `dataset/password.txt` - 密码文件（加密存储）

## 数据文件格式

联系人文件格式：
```
ID|姓名|工作单位|电话号码|E-mail地址
```

示例：
```
1|张三|北京科技公司|13800138000|zhangsan@example.com
2|李四|上海软件公司|13900139000|lisi@example.com
```
