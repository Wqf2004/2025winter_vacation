# 电话簿管理系统

**✅ 已完成** - 包含所有十个功能，并提供三个不同技术栈的版本

## 项目概述

电话簿管理系统（Phonebook Management System）是一个功能完整的联系人信息管理工具，支持联系人添加、查询、修改、删除、排序、搜索、文件存储、密码验证和数据纠错等功能。本项目提供了三个版本，满足不同使用场景的需求。

## 版本对比

| 特性                 | C语言控制台版 | PyQt5桌面版    | Streamlit网页版        |
| -------------------- | ------------- | -------------- | ---------------------- |
| **界面类型**   | 命令行界面    | 桌面GUI应用    | Web网页应用            |
| **技术栈**     | C语言         | Python + PyQt5 | Python + Streamlit     |
| **安装复杂度** | 需编译        | 需安装PyQt5    | 需安装Streamlit        |
| **使用方式**   | 本地终端运行  | 本地桌面运行   | 浏览器访问（支持远程） |
| **界面美观度** | 基础文本界面  | 现代化桌面界面 | 现代化网页界面         |
| **跨平台**     | 需重新编译    | 完全支持       | 完全支持               |
| **远程访问**   | 不支持        | 需远程桌面     | 原生支持               |
| **适用场景**   | 学习研究      | 个人使用       | 局域网/云端部署        |

## 功能需求实现

本项目完整实现了题目要求的所有十项功能：

✅ ① 加入一个新电话号码  
✅ ② 删除一个电话号码  
✅ ③ 显示保存的所有电话号码  
✅ ④ 修改功能  
✅ ⑤ 排序功能（按电话号码排序、按姓名字母序排列）  
✅ ⑥ 查询功能（按人名查询电话号码、按电话号码查询人名）  
✅ ⑦ 保存信息到文件  
✅ ⑧ 从文件读取信息  
✅ ⑨ 具有登录功能和密码加密功能  
✅ ⑩ 具有纠错功能（手机号码和E-mail地址正确性检查）

## 版本一：C语言控制台版

### 基本信息

- **目录位置**：`c_console/`
- **主程序**：`main.c`
- **数据文件**：`dataset/contacts.txt`、`dataset/password.txt`
- **默认密码**：`admin123`

### 编译运行

#### Windows
```bash
compile.bat
```

#### Linux/Mac
```bash
chmod +x compile.sh
./compile.sh
```

### 功能特点

1. 完整的命令行菜单界面
2. 密码加密存储（异或加密）
3. 电话号码格式验证
4. E-mail地址格式验证
5. 重复电话号码检查
6. 文件数据持久化
7. 快速排序功能（qsort）

### 数据文件格式

联系人文件格式：
```
ID|姓名|工作单位|电话号码|E-mail地址
```

示例：
```
1|张三|北京科技公司|13800138000|zhangsan@example.com
2|李四|上海软件公司|13900139000|lisi@example.com
```

## 版本三：Streamlit网页版

- **主程序**：`app.py`
- **技术栈**：Python 3.7+ + Streamlit + Pandas
- **默认密码**：`admin123`

### 安装运行

```bash
pip install -r requirements.txt
streamlit run app.py
# 或双击 run.bat（Windows）
```

启动后浏览器自动打开 `http://localhost:8501`

### 局域网部署

```bash
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

其他设备可通过 `http://你的IP地址:8501` 访问

### 云端部署

支持以下平台免费部署：

- **Streamlit Cloud**：连接GitHub即可

## 公共功能说明

### 数据验证规则

#### 电话号码验证
- 长度：7-15位
- 允许字符：数字、连字符(-)、空格
- 自动检查重复号码

#### E-mail验证
- 长度：5-99位
- 必须包含且仅包含一个 @ 符号
- @ 符号后必须包含点号
- 格式：xxx@xxx.xxx

### 数据文件说明

三个版本共享同一套数据文件（位于 `dataset/` 目录）：

- `contacts.txt` - 联系人信息
- `password.txt` - 登录密码文件

**注意**：密码文件使用异或加密存储，保障安全性。

### 排序功能

- 按姓名字母序排序：使用字典序
- 按电话号码排序：使用字典序

### 查询功能

- 按人名查询：支持模糊匹配（关键词搜索）
- 按电话号码查询：支持模糊匹配（关键词搜索）
