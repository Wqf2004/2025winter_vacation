# 电话簿管理系统 - PyQt5桌面版

## 功能说明

1. **添加联系人** - 通过对话框输入联系人信息，自动验证格式
2. **删除联系人** - 选择联系人后删除，有确认提示
3. **修改联系人** - 修改联系人的各项信息
4. **排序功能** - 按姓名字母序或按电话号码排序
5. **查询功能** - 实时搜索，支持按姓名或电话号码搜索
6. **保存数据** - 将联系人信息保存到文件
7. **修改密码** - 修改系统登录密码
8. **数据验证** - 自动验证电话号码和E-mail格式

## 数据验证规则

### 电话号码验证
- 长度：7-15位
- 允许字符：数字、连字符(-)、空格
- 自动检查重复号码

### E-mail验证
- 必须包含 @ 符号
- @ 符号后必须包含点号
- 基本格式验证

## 安装运行

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

或双击 `run.bat`（Windows）

## 默认密码

- 默认密码：`admin123`
