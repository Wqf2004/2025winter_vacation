# 销售系统 - C语言控制台版

## 功能说明

### 管理员功能
1. 修改密码
2. 用户管理：添加、修改、查询、删除用户
3. 商品管理：添加、修改、查询、删除商品
4. 销售报表：日报表、月报表、商品销售量报表、销售员业绩报表
5. 退出系统

### 店长功能
1. 修改密码
2. 商品管理：添加、修改、查询、删除商品
3. 销售报表：日报表、月报表、商品销售量报表、销售员业绩报表
4. 退出系统

### 销售员功能
1. 商品浏览和销售
2. 销售报表：日报表、月报表（仅显示自己的记录）
3. 退出系统

## 编译运行

### Windows
```bash
compile.bat
main.exe
```

### Linux/Mac
```bash
chmod +x compile.sh
./compile.sh
./main
```
