# 销售系统 - Streamlit网页版

## 功能说明

### 所有角色通用功能
- 登录验证
- 日报表查看
- 月报表查看

### 管理员功能
- 商品浏览
- 商品销售量报表
- 销售员业绩报表

### 店长功能
- 商品浏览
- 商品销售量报表
- 销售员业绩报表

### 销售员功能
- 商品浏览
- 商品销售
- 日报表（仅显示自己的记录）
- 月报表（仅显示自己的记录）

## 安装运行

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行
```bash
run.bat
# 或手动运行
streamlit run app.py
```

## 使用说明

1. 登录系统（使用默认账户）
2. 根据角色选择对应功能
3. 销售员可以在"销售商品"页面进行销售操作
4. 所有角色都可以查看报表

## 默认账户

- **管理员**: admin / admin123
- **店长**: manager / manager123
- **销售员**: sales1 / sales123
- **销售员**: sales2 / sales456
