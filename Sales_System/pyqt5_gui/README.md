# 销售系统 - PyQt5桌面版

## 功能说明

PyQt5版本提供图形化界面，支持所有角色的完整功能。

### 功能模块
- 登录验证
- 密码修改
- 用户管理（管理员）
- 商品管理（管理员、店长）
- 商品销售（销售员）
- 各类报表显示

## 安装运行

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行
```bash
run.bat
# 或手动运行
python main.py
```

## 默认账户

- **管理员**: admin / admin123
- **店长**: manager / manager123
- **销售员**: sales1 / sales123
- **销售员**: sales2 / sales456

## 注意事项

PyQt5版本的核心框架已完成，完整的GUI界面可以参考Streamlit版本的数据管理逻辑进行开发。
