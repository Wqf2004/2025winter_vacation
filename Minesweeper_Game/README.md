1. 创建一个全局数组map[][]，对数组进行操作，默认雷为-1，如果想要绘制10*10的布局，考虑到后续计算9宫格时，需要建立 12*12的二维数组
2. 游戏初始化函数，绘制画布，进行数组的填充。GameInit()
	建立一个二维数组，默认内容为0；
	随机行列（要在1-10行，1-10列），任意选定10个位置为雷，赋值-1；
	循环遍历10*10个位置内容（从第二行第二个开始），绘制9宫格，依次计算9宫格内雷的个数，进行数据填充；
 
	图片加载函数

## 项目概述

扫雷是一款经典的逻辑推理游戏，本程序实现完整的游戏逻辑和界面。

## 功能需求

- 雷区生成
- 雷点标记
- 计时功能
- 剩余雷数
- 游戏难度选择
- 排行榜
