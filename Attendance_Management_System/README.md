# 项目二：出勤管理系统

**✅ 已完成**

## 项目概述

出勤管理系统用于企业职工的出勤管理，支持打卡记录、出勤计算、请假管理、统计分析等功能。系统通过读卡机记录职工上班时间，自动计算迟到次数、旷工天数，并根据企业规定计算奖罚金额。

## 业务规则

### 1. 读卡机规定

- 每个职工每天早晨上班时在读卡机上刷卡
- 正点上班时间为早上8点，早上9点准时关闭读卡机
- 最多支持500名职工

### 2. 奖罚规定

**旷工扣款：**
- 每旷工一天扣发当月工资的10%
- 旷工定义：当日未刷卡或迟到1小时及以上

**迟到扣款：**
- 迟到10分钟以下：迟到1次
- 迟到10-20分钟：迟到2次
- 以此类推，每迟到一次扣当月工资0.5%

**请假优惠：**
- 女职工每月可请假3天（免扣工资）
- 55岁以上男职工每月可请假2天（免扣工资）
- 生日所在月份可请假1天（免扣工资）
- 超过允许天数：每请假1天扣当月工资5%

**全勤奖励：**
- 全勤职工一次性奖励400元（有请假就不算全勤）

## 数据结构

### 职工基本信息（EmpBasic）
- 职工编号（唯一）
- 姓名
- 性别
- 出生日期（格式：YYYYMMDD）
- 职务级别
- 所在部门

### 出勤记录（Attendance）
- 职工编号
- 迟到次数
- 请假天数
- 旷工天数

### 打卡记录（PunchIn）
- 职工编号
- 时（整数）
- 分（整数）
- 秒（整数）

## 核心算法

### 迟到次数计算
```
迟到时间 = (时*3600 + 分*60 + 秒) - 8*3600
如果 迟到时间 > 0:
    迟到分钟数 = 迟到时间 / 60
    迟到次数 = ceil(迟到分钟数 / 10)
```

### 允许请假天数计算
```
allowed = 0
if 性别 == "女": allowed += 3
if 性别 == "男" and 年龄 >= 55: allowed += 2
if 当前月份 == 生日月份: allowed += 1
```
